# Laser Defender

Shoot 'em up game from a **GameDev.tv** course

### Instructions to download and play

1. Click `Game` folder
2. Click `Laser Defender Game.zip`
3. Select `Download`
4. Unzip file
5. In unzipped folder doubleclick `Laser Defender.exe`
6. Enjoy!
